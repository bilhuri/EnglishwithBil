1. Sentence Formula

Bahasa Inggris itu terstruktur. Jangan pernah menukar posisi Subjek dan Kata Kerja.

English is structured. Never swap the position of the Subject and the Verb.

// Rumus Dasar (Basic Formula)
Subject + Verb + Object

// Contoh Benar:
-> I eat rice.
// Contoh Salah (Error):
-> Eat rice I.

2. Subject & Pronouns

Pelaku dibagi menjadi dua kelompok. Hafalkan ini karena aturannya berbeda nanti.

Actors are divided into two groups. Memorize this because the rules are different later.

// Kelompok A (Group A)
I, You, We, They

// Kelompok B (Group B - Spesial)
He, She, It (Orang Ketiga Tunggal)

3. To Be (Nominal Logic)

Gunakan To Be (Jembatan) HANYA jika kalimatmu tidak punya kata kerja aksi. Kita harus lengkap dengan bentuk positif, negatif, dan tanya-nya.

Use To Be (Bridge) ONLY if your sentence has no action verb. We must complete the positive, negative, and question forms.

// Logika: IF (Tidak Ada Aksi) -> USE To Be
// Pasangan (Mapping): I->am, He/She/It->is, You/We/They->are

(+) Positif: Subject + To Be + Adj/Noun
-> She is smart.

(-) Negatif: Subject + To Be + NOT + Adj/Noun
-> She is not smart.

(?) Tanya: To Be + Subject + Adj/Noun ?
-> Is she smart?

4. Simple Present

Logika untuk fakta dan kebiasaan. Perhatikan sifat "manja" dari He, She, It.

Logic for facts and habits. Notice the "needy" nature of He, She, It.

// Cek Subjek (Check Subject):
IF Subject = He / She / It
THEN Verb + s/es

IF Subject = I / You / We / They
THEN Verb (Normal)

// Contoh:
-> She eats pizza.
-> They eat pizza.

5. Negative Logic

Untuk bilang "Tidak", kita butuh bantuan (Do/Does).

To say "No", we need help (Do/Does).

// Rumus Negatif:
Subject + Don't / Doesn't + Verb-1

// Logika Pemilihan:
I, You, We, They -> Don't
He, She, It -> Doesn't

// Peringatan (Warning):
-> She doesn't eat. (✅ Verb balik normal)
-> She doesn't eats. (❌ Error: Double S)

6. Question Logic

Untuk bertanya, tarik kata bantu ke paling depan.

To ask, pull the helper word to the very front.

// Rumus Tanya (?):
Do / Does + Subject + Verb-1 ?

// Contoh:
-> Do you like coffee?
-> Does she like coffee?

7. Adjectives Position

Hukum M-D (Menerangkan-Diterangkan). Kata sifat selalu DI DEPAN benda.

Adjectives always come BEFORE the noun.

// Urutan (Order):
[Adjective] + [Noun]

// Contoh:
-> Smart Boy (✅)
-> Boy Smart (❌ Error)

8. Demonstratives

Logika menunjuk benda berdasarkan Jarak dan Jumlah.

Logic for pointing at things based on Distance and Quantity.

// Dekat (Near):
This -> Satu Benda (1)
These -> Banyak Benda (>1)

// Jauh (Far):
That -> Satu Benda (1)
Those -> Banyak Benda (>1)

9. Possessives

Ada dua jenis kepemilikan. Yang pertama (Adjective) harus diikuti benda. Yang kedua (Pronoun) tidak perlu benda lagi.

There are two types of ownership. The first (Adjective) must be followed by a noun. The second (Pronoun) does not need a noun.

// Jenis 1: Adjective (Wajib ada benda)
My, Your, Her, His, Its, Our, Their
-> My car is red.

// Jenis 2: Pronoun (Tidak perlu benda)
Mine, Yours, Hers, His, Ours, Theirs
-> This car is mine. (✅)

10. Time Prepositions

Logika Segitiga Waktu: Dari yang terluas hingga yang paling spesifik.

Time Pyramid Logic: From broadest to most specific.

IN -> Tahun, Bulan, Abad (Paling Luas)
// in 1990, in October

ON -> Hari, Tanggal (Sedang)
// on Sunday, on 17 August

AT -> Jam (Sangat Spesifik)
// at 7 PM, at night
