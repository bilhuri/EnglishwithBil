1. Conditional Type 0

Logika Fakta/Kebenaran Universal. Jika terjadi, maka hasilnya pasti terjadi.

Logic for Universal Truths/Facts. If the condition occurs, the result always occurs.

// Rumus Fakta:
IF Present Simple, THEN Present Simple

-> If you heat water, it boils.

2. Conditional Type 1

Logika Kemungkinan Nyata (Real Possibility) di Masa Depan. Paling sering dipakai.

Logic for Real Possibility in the Future. Most commonly used.

// Rumus Nyata:
IF Present Simple, THEN Future Simple (Will)

-> If I find silver, I will sell it.

3. Conditional Type 2

Logika Tidak Nyata (Unreal) di Masa Kini. Sesuatu yang kita harap terjadi, tapi tidak mungkin terjadi sekarang.

Logic for Unreal Present. Something we wish for, but is impossible right now.

// Rumus Khayalan (Unreal Present):
IF Past Simple, THEN Would + V1

-> If I were a millionaire, I would buy a yacht.
// Note: Selalu gunakan 'were' untuk semua subjek (I, He, She) di Type 2.

4. Conditional Type 3

Logika Penyesalan (Regret) di Masa Lampau. Sudah tidak bisa diubah lagi.

Logic for Past Regret. Something that cannot be changed anymore.

// Rumus Penyesalan (Unreal Past):
IF Past Perfect, THEN Would Have + V3

-> If you had studied, you would have passed.

5. Gerunds (Verb as Noun)

Mengubah kata kerja menjadi kata benda. Wajib dihafalkan setelah kata kerja tertentu (misal: *enjoy*, *mind*, *stop*).

Turning a verb into a noun. Mandatory after certain verbs (e.g., *enjoy*, *mind*, *stop*).

// Formula:
Verb + V-ing

// Contoh:
-> Coding is fun. (V-ing sebagai Subjek)
-> I enjoy watching movies. (V-ing sebagai Objek)

6. Infinitives (To + Verb)

Bentuk kata kerja murni. Wajib dihafalkan setelah kata kerja tertentu (misal: *want*, *need*, *agree*) dan setelah kata sifat.

Pure verb form. Mandatory after certain verbs (e.g., *want*, *need*, *agree*) and after adjectives.

// Formula:
Verb + to + V1

// Contoh:
-> I want to buy gold. (Setelah 'want')
-> It is easy to understand. (Setelah kata sifat)

7. Participles (V-ing / V-ed as Adjectives)

Mengubah kata kerja menjadi kata sifat. Ini memadatkan kalimat.

Turning a verb into an adjective. This compacts sentences.

// V-ing (Active/Sedang): The boring lesson.
// V-ed (Passive/Selesai): The broken glass.

8. Active vs Passive Participles

Jangan salah, "V-ing" bukan selalu Continuous. Ini menunjukkan apakah benda melakukan atau menerima aksi.

Don't confuse them. V-ing isn't always Continuous. It shows if the thing is doing or receiving the action.

V-ing (Active): Sesuatu yang melakukan aksi.
-> I read a boring book. (Buku itu membuat bosan).

V-ed (Passive): Sesuatu yang menerima aksi.
-> I am bored. (Saya merasa bosan).

9. Subjunctive (Hypothetical Wish)

Logika Formal untuk permintaan, saran, atau keharusan. Wajib menggunakan V1 murni tanpa s/es/ed.

Formal Logic for demands, suggestions, or necessity. Must use pure V1 (no s/es/ed).

// Rumus:
S + Verb Suggestion + THAT + S + V1 Murni

-> I insist that he be here. (Bukan 'is')
-> I recommend that she study harder. (Bukan 'studies')

10. Inversion (For Emphasis)

Logika penekanan kalimat secara formal (TOEFL/IELTS). Memindahkan Auxiliary/Modal ke depan subjek.

Formal logic for sentence emphasis (TOEFL/IELTS). Moving the Auxiliary/Modal before the subject.

// Contoh Inversion:
Normal: I have never seen such beauty.
Inverted: Never have I seen such beauty.

// Digunakan setelah kata negatif:
Not only, Seldom, Rarely, etc.
