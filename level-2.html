1. Present Continuous

Logika untuk kejadian yang SEDANG berlangsung detik ini juga (Live Action).

Logic for events happening RIGHT NOW (Live Action).

// Rumus "Sedang" (Now):
Subject + To Be + Verb-ing

// Contoh:
-> I am studying.
-> She is eating.

// Logic Check:
IF Time = Now / At the moment
USE V-ing

2. Stative Verbs (Warning)

Kata kerja perasaan/pikiran HARAM hukumnya dijadikan "Sedang" (V-ing). Hati-hati!

Feelings/Thoughts verbs are FORBIDDEN in Continuous (V-ing). Be careful!

Forbidden List: Love, Know, Hate, Understand, Believe.

// Contoh Salah (Error):
-> I am loving you. (❌ Alay/Grammar Error)
-> I am knowing him. (❌)

// Koreksi (Pakai Simple Present):
-> I love you. (✅)
-> I know him. (✅)

3. Simple Past

Logika Sejarah. Kejadian sudah selesai di masa lalu dan waktunya jelas.

History Logic. Event finished in the past and time is specific.

// Rumus Masa Lalu:
Subject + Verb-2

// Time Signal Wajib:
Yesterday, Last Night, 2 Days Ago.

// Contoh:
-> I bought silver yesterday.

4. Regular vs Irregular

Perubahan kata kerja masa lalu (V2) ada dua pola. Ini hafalan wajib.

Past verb changes (V2) have two patterns. Essential memorization.

// 1. Regular (Beraturan):
Logic: Cukup tambah -ed
-> Play -> Played
-> Cook -> Cooked

// 2. Irregular (Acak):
Logic: Berubah total / Tetap
-> Buy -> Bought
-> Go -> Went
-> Cut -> Cut

5. Past Continuous

Logika "Sedang" tapi di masa lalu. Spesifik di jam tertentu kemarin.

Logic "In Progress" but in the past. Specific time yesterday.

// Rumus:
Subject + Was/Were + Verb-ing

// Konteks:
-> I was sleeping at 8 PM last night.

6. Simple Future (Will)

Masa depan spontan. Janji, prediksi, atau keputusan dadakan.

Spontaneous future. Promises, predictions, or sudden decisions.

// Rumus Spontan:
Subject + Will + Verb-1

// Contoh:
A: It's hot.
B: I will open the window. (Dadakan)

7. Future (Going To)

Masa depan terencana. Sudah dipikirkan sebelumnya.

Planned future. Thought about it before.

// Rumus Rencana:
S + To Be + Going To + Verb-1

// Contoh:
-> I am going to visit Aceh next month.
// (Tiket sudah dibeli / Rencana matang)

8. Present Perfect vs Simple Past

Ini adalah *error* paling umum. Kunci: Simple Past wajib ada WAKTU JELAS. Present Perfect WAKTU TIDAK PENTING (atau baru saja).

This is the most common error. Key: Simple Past must have a CLEAR TIME. Present Perfect: TIME DOES NOT MATTER (or recently).

PRESENT PERFECT: S + Have/Has + V3
// Fokus pada HASIL: -> I have finished coding. (Hasil: Kode selesai sekarang.)

SIMPLE PAST: S + V2
// Fokus pada WAKTU: -> I finished coding yesterday. (Waktu spesifik: kemarin.)

9. Time Markers

Penanda waktu adalah kunci detektif untuk menentukan Tenses mana yang dipakai.

Time markers are detective keys to determine which Tense to use.

// Keyword Masa Lalu (Use V2):
Yesterday, ...Ago, Last...

// Keyword Masa Depan (Use Will):
Tomorrow, Next..., Soon

// Keyword Rutinitas (Use V1):
Every day, Usually, Always

10. Timeline Summary

Ringkasan logika garis waktu dalam satu pandangan.

Timeline logic summary in one view.

Past (Sejarah) -----------> V2
Now (Sedang) -----------> V-ing
Future (Nanti) -----------> Will
Result (Hasil) -----------> Have + V3
